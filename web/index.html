Bluetooth RC Car Controller

This repository contains a Kotlin + Jetpack Compose Android app and an Arduino firmware (.ino) to control a 4WD RC car via HCâ€‘05 Bluetooth (SPP). Deploying this page provides a stable URL for documentation and downloads; the controller itself runs on Android and communicates with your car's Arduino.

Build the Android app

  1. Open the project in Android Studio (Giraffe+).
  2. Use a device running Android 7.0+ (API 24+).
  3. Build and run the app module.

Main files: app/src/main/java/com/bd/rc/

Flash Arduino firmware

  1. Open arduino/rc_car.ino in Arduino IDE 2.3.2+.
  2. Select Arduino Uno, set baud 9600, and upload.
  3. Wire per the hardware truth in the README.

Deployed via Vercel. App logs tagged RC_CAR.
